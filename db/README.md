# Database Setup

MySQL database hosted on AGH server (`mysql.agh.edu.pl`). **Access requires AGH network or VPN.**

Database credentials are available in [panel.agh.edu.pl](https://panel.agh.edu.pl).

## First-time Setup

1. Log into [phpMyAdmin](https://mysql.agh.edu.pl/phpMyAdmin)
2. Select the `argo` database
3. Go to the **SQL** tab
4. Paste the contents of `schema.sql` and execute

## Files

| File | Tracked | Purpose |
|---|---|---|
| `schema.sql` | ✅ | Table definitions — run once on setup or when structure changes |
| `db_config.php` | ✅ | DB host, name, user — no secrets |
| `db_psswd.php` | ❌ | Defines `DB_PASS` — gitignored, generated by GitHub Actions in production |
| `db.php` | ✅ | `get_pdo()` — reusable PDO connection function |
| `migrate.php` | ✅ | One-time script to import existing posts into DB |

## Local Development

`db_psswd.php` is gitignored. Create it manually for local dev:

```php
<?php define('DB_PASS', 'your_password'); ?>
```
## Production Deploy

`db_psswd.php` is generated automatically by GitHub Actions from the `db_psswd_PHP` secret before rsync. No manual step needed.

## Schema Changes

Edit `schema.sql` and re-run the relevant statements in phpMyAdmin. For destructive changes (dropping columns, renaming) do it manually and carefully — there is no rollback.
